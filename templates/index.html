Tapo Camera Viewer (gortsplib)

⚠️ H.264 Decoder Implementation Required

This version uses gortsplib but requires a complete H.264 decoder implementation. The current version is a framework that needs either:

  • FFmpeg bindings (github.com/u2takey/ffmpeg-go)
  • Pure Go H.264 decoder (github.com/yapingcat/gomedia)
  • WebRTC implementation with proper video decoding

For immediate functionality, consider keeping FFmpeg for decoding while using gortsplib for RTSP handling.
